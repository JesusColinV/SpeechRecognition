Funciones

Reconocimiento de voz

Presiona el microfono para poder leer lo que se está diciendo

Cambia el tema de la aplicación

Una vez que se activa el microfono se habilitan los siguientes comandos

Comando de Voz Acción
"Iniciar Cambio de Tema" Habilita la Estrategia para el cambio de Tema
"Finalizar Cambio de Tema" Finaliza la Estrategia para el cambio de Tema

Nombres de tema disponibles: "púrpura", "azul", "rosa" and "verde"

Tablero de comunicación

Seleccionado el botón de tarjetas, podrás acceder a un catalogo de necesidades que generarán voz, o puedes repetir los siguientes comandos

Comando de Voz Acción
"Yo necesito ayuda" Habilita la Estrategia para desplegar catalogo de necesidades
"Ya no necesito ayuda" Finaliza la Estrategia de necesidades
